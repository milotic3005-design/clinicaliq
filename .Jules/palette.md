## 2026-04-27 - Accessible Clear Button and Loading Layout Stability
**Learning:** When displaying a loading spinner in an input field with multiple icons, replacing the left-side 'Search' icon with a spinner provides better layout stability than placing it on the right. Keeping the right side stable allows for consistent placement of a clear button, ensuring it doesn't shift or disappear during async operations. Also, a clear button in an input must have proper ARIA labels, focus-visible outlines, and predictable re-focusing of the input after clearing to ensure full keyboard accessibility.
**Action:** Always replace the primary static icon with a spinner when loading to avoid right-side layout jumps. Always pair clear buttons with input re-focus and focus-visible styling.
